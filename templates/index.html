About This Tool

This tool predicts various molecular properties using graph neural networks. The current model is trained on the ESOL dataset and optimized for predicting log solubility in mol/L.

You can input either:

  • A SMILES string representing the molecular structure
  • A common name of a chemical compound (which will be converted to SMILES)

Future updates will include support for additional properties from the MoleculeNet collection.
